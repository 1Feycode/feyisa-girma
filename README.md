# Feyisa Girma — Portfolio

Personal portfolio of Feyisa Girma, Network Engineer & Full-Stack Developer.

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS (dark/light theme)
- Framer Motion
- EmailJS (contact form)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Live at: https://feyisa-girma.vercel.app/
